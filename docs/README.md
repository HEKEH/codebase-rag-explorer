# Docs Directory Guide

This directory is organized by document purpose and lifecycle stage.

## Structure

编号子目录约定如下（当前仓库内均已存在；若后续增加 `00-overview/`、`99-archive/` 等，仍沿用同一编号体系）。

- `01-product/`: product requirement and scope definitions
- `02-technical/`: architecture and technical design
- `03-planning/`: roadmap and remediation plans
- `04-implementation/`: implementation progress and issue tracking
- `05-quality/`: acceptance criteria, reports, and evaluation assets
- `06-operations/`: runtime logging and operations notes
